Frontend
1. Цель
Создать клиентскую часть проекта «Tinder для домашки» — веб-приложение, позволяющее студентам:
зарегистрироваться и авторизоваться;
создавать и просматривать карточки с заданиями;
свайпать карточки и находить совпадения.

2. Основной функционал
Авторизация и регистрация
Страницы: Login, Register
Валидация полей и отображение ошибок
Хранение данных пользователя
Профиль пользователя
Страница Profile: отображение информации о пользователе
Возможность редактирования профиля (имя, факультет, фото, контакты)
Просмотр своих карточек
Карточки
Страница Feed с карточками заданий
Возможность свайпа: лайк / пропуск
Создание и редактирование карточек с заданиями (форма: предмет, тип работы, описание)
Лайки
Список совпадений (имя, аватар, предмет)

3. Требования к архитектуре
Фреймворк: React
Маршрутизация: React Router
Версия Node.js: 18+

4. Требования к UX/UI
Дизайн в духе Tinder: карточки, свайпы, лёгкий интерфейс
Простой и понятный UX: минимум действий для ключевых сценариев
Цветовая схема: нейтральная, без отвлекающих элементов
